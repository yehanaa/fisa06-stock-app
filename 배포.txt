# Deploy
- 개발자의 컴퓨터에서 설치한 패키지, 필요한 파일들을
실제 서비스를 하고 있는 컴퓨터로 옮기는 과정


1 ) 필요한 패키지를 현재 상태 그대로 파일로 저장
- 파이썬 패키지는 requirments.txt 라는 파일명을 보통 사용
pip freeze > requirments.txt

2 ) 쓰지 않을 불필요한 파일들/남들에게 공개하지 말아야 할 파일을 제거
- .gitignore 라는 파일에 불필요하거나 공개되면 안 될 파일의 리스트를 작성
- 벌써 .gitignore 없이 git add .을 했다면
- git rm --cached *
- 이후 git add . -> gut gimmit -m "메세지" -> git push -u origin main
* 주의사항 : 그럼에도 불구하고 이전 커밋에 남아있음


3 ) 원격 서버가 확인할 수 있는 레퍼지토리에 올림
